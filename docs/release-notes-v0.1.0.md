# Conceal FlClash Android v0.1.0

Initial Conceal FlClash Android release by YChaiyi.

## Included

- Renamed Android app to `Conceal FlClash Android`.
- Changed APK package name to `com.github.ychaiyi.conceal_flclash`.
- Preserved FlClash's built-in Android `VpnService` TUN path.
- Added `Conceal FlClash TUN Helper` for SukiSU Ultra/Magisk.
- Removed dependency on upstream Firebase build configuration.
- Updated README and release notes for the Conceal fork.

## Install

1. Install the APK.
2. Open the app once, select a profile, enable TUN/VPN mode, and approve Android VPN consent.
3. Install the helper ZIP in SukiSU Ultra.
4. Reboot or use the module action button to start/stop the TUN service.
